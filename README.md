# NBAdata

## Описание 
Андроид приложение позволяет получить данные nba игроков, команд и игр за 2019.

## Технологии 
Клиентская часть: android studio kotlin. Запрос api.

- clean,<br/>
- viewModel,<br/>
- mvvm, <br/>
- di hilt, <br/>
- navigation, <br/>
- databinding, <br/>
- scenes transition with lottie,<br/>
- animation start,<br/>
- recyclerViews,<br/>
- viewPager2,<br/>
- dialog,<br/>
- retrofit<br/>
